## Start projektu

- [ ]  Utwórz nowy projekt Laravel 12 w HERD
    
    ```bash
    laravel new 4app-portfolio
    
    ```
    

Skonfiguruj .env (DB, MAIL, APP_URL)

Zainstaluj Filament v4 i utwórz konto admina

```
composer require filament/filament:"^3.0"
php artisan make:filament-user

```

